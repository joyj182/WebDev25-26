Web Technologies

HTML

HyperText markup language. HTML brings in the elements to the web page. For example elements like headings, paragraphs, and images

CSS

Cascading Style Sheets. CSS allows you to style/decorate the elements on the web page. For example color, font, backgrounds, and borders.

JavaScript

JavaScript allows you to control the behavior of the elements on the webpage. JavaScript can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.
